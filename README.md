# 📩 SMS Spam Detection System

A Machine Learning-based SMS Spam Detection web application that classifies messages as Spam or Safe using Natural Language Processing (NLP) and a Multinomial Naive Bayes classifier.

## 🚀 Features

- Spam and Safe Message Detection
- TF-IDF Vectorization
- Multinomial Naive Bayes Classifier
- Spam Risk & Safe Message Confidence Scores
- Suspicious Keyword Detection
- Interactive Streamlit Web Interface

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLTK
- Pandas
- NumPy

## 📊 Dataset

SMS Spam Collection Dataset

- ~5,169 SMS messages
- Ham (Safe) and Spam classes

## 📈 Model Performance

- Accuracy: 97.09%
- Precision: 100%
